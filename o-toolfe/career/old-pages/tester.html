Job Overview

Join us as a Software Tester Intern, where you'll collaborate with our QA team to test software applications, identify bugs, and contribute to quality improvement. Gain hands-on experience, execute test cases, and participate in creating test documentation during this internship. This is an opportunity to develop practical skills in software testing methodologies within a dynamic work environment.

Responsibilities:

  • Collaborate with the QA team to learn software testing methodologies and practices.
  • Assist in creating and executing manual test cases and test plans based on software requirements.
  • Perform functional, regression, and user acceptance testing.
  • Document and report software defects and issues discovered during testing.
  • Work closely with developers to reproduce and troubleshoot identified issues.
  • Contribute to improving testing processes and methodologies.

Requirements :

  • Recent graduate in Computer Science, Engineering, or related field.
  • Possession of a certification in software testing or completion of relevant coursework.
  • Strong interest in software testing methodologies and practices.
  • Basic understanding of software development life cycle (SDLC).
  • Familiarity with bug tracking tools and test management software (preferred).
  • Excellent analytical and problem-solving skills.
  • Strong attention to detail and ability to work well within a team environment.

Benefits :

  • Hands-on experience in testing methodologies.
  • Stipend of up to ₹5000 for the 3-month internship.
  • Potential for future career opportunities based on performance.
  • Opportunity to contribute to real-world software projects.
